# 00 - Inbox

Zona de captura rápida para material sin procesar.

## Uso

- Notas rápidas que aún no tienen destino
- Ideas a investigar
- Material temporal antes de mover a `sources/`

## Regla

> Lo que está en Inbox más de 7 días debe ir a `sources/` o borrarse.

## Relacionado

- `sources/` — Raw sources organizados
- `notes/sistema/ai-second-brain-claude-obsidian` — Sistema completo
