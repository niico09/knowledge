# Onboarding — Primeros 3 Prompts

## Prompt 1 — Con docs propios

Crear notebook, subir fuentes, curriculum map, registrar en Notion.

## Prompt 2 — Sin docs (Deep Research)

Arrancar desde cero. Deep Research busca fuentes, crea notebook, curriculum map.

## Prompt 3 — Generar SKILL.md

Desde notebook existente → SKILL.md listo para Claude Code.

## Checklist

- [ ] Notebook nuevo en NotebookLM
- [ ] Entrada en Notion > Notebooks Activos
- [ ] Prompt en Prompts Library
- [ ] SKILL.md listo para probar
