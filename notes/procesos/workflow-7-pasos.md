# Workflow Overview — Los 7 Pasos

## Los 7 Pasos

1. **Estrategia de carga** — Subir toda la bibliografía de la unidad
2. **Mapeo curricular** — 5 conceptos core + secuencia
3. **Plan de clases** — Day-by-day lesson plan
4. **Materiales para estudiantes** — Guías, flashcards, quizzes
5. **Audio Overview** — Podcast resumen para pre-lectura
6. **Deep Research** — Revisiones de literatura (70% menos tiempo)
7. **Google Classroom** — Distribución instantánea

## Tiempo ahorrado

| Paso | Ahorro |
|------|--------|
| 2 | Semanas → horas |
| 3 | 2 semanas → tarde |
| 4 | 1 fin de semana → minutos |
| 6 | 3 meses → 70% menos |
