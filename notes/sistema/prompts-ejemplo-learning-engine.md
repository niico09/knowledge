# Prompts de Ejemplo — Learning Engine

## Caso 1 — Aprender algo nuevo con docs propios

```
Quiero aprender Zustand para manejo de estado en React.
Te adjunto la documentación oficial (PDF).
Armame el curriculum map y registralo en Notion.
```

## Caso 2 — Aprender desde cero sin docs propios

```
Quiero aprender CUDA programming desde cero, no tengo docs propios.
Arrancá con Deep Research y construí el notebook.
```

## Caso 3 — Consulta de profundización sobre notebook existente

```
Tengo el notebook chakra-ui. Quiero entender bien cómo funciona
el sistema de tokens en Chakra UI v3 con Emotion.
Usá el patrón de triangulación y citá las fuentes.
```

## Caso 4 — Derivar al Asset Generator

```
Ya aprendí lo suficiente sobre React Query. Ahora quiero
generar un SKILL.md y rules para mi proyecto.
¿Puedo hacer eso acá o tengo que ir al Asset Generator?
```
