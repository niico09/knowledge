# NotebookLM Knowledge Base

Base de conocimiento construida a partir del workflow documentado por facultad de NYU, Stanford, Case Western, ASU y Northeastern.

## Estructura

| Sección | Descripción |
|---------|-------------|
| Workflow Overview | Los 7 pasos del workflow completo con prompts |
| Prompts Library | Base de datos de prompts filtrable |
| Notebooks Activos | Registro de todos tus notebooks |
| Experimentos & Resultados | Log de lo que probaste y aprendiste |
| Integraciones | MCP, Claude Code, Google Classroom |

## Quick Start

1. Empezar por **Workflow Overview** para entender el sistema completo
2. Usar **Prompts Library** para copiar prompts listos
3. Registrar notebooks en **Notebooks Activos**
4. Documentar lo aprendido en **Experimentos & Resultados**
5. Usar **Asset Generator v3** para generar assets
