# Radar IA para Código — 2 Abr 2026

## Papers destacados

### Ask or Assume? — Clarification-Seeking in Coding Agents
Mejora tasa de resolución de issues detectando ambigüedad subespecificada y buscando clarificaciones proactivamente.

### SWE-PRBench — Benchmarking AI Code Review Quality
Revela brechas significativas entre rendimiento humano y de modelos. Contexto completo del repo degrada rendimiento por dilución de atención.

### Rethinking Agent-Generated Tests
Evidencia empírica: escribir tests aporta beneficio marginal para resolución de issues. Agentes prefieren depuración por observación directa.

### MEnvAgent — Polyglot Environment Construction
Framework multi-lenguaje que automatiza construcción de entornos Docker para SE.

### ABC-Bench — Agentic Backend Coding in Real-World Development
224 tareas en 8 lenguajes y 19 frameworks. Even los mejores modelos fallan en tareas backend holísticas.

## Tendencias

1. **Nueva oleada de benchmarks en entornos reales** — Ejecución real, deployment containerizado, hardware físico
2. **Agentes cuestionan sus estrategias de testing** — Testing asistido por IA tiene límites estructurales
3. **Especialización vertical** — El valor está en adaptación a dominios/lenguajes/stacks concretos
