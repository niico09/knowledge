# Pricing — API Directo vs Subscription

## El Benchmark

| Método | Costo | Notas |
|--------|-------|-------|
| **API directo** | ~$500/mo | Mismas tareas que via Claude Code |
| **Pro ($20/mo)** | $20/mo | Bueno para empezar |
| **Max 5x ($100/mo)** | $100/mo | Suficiente para la mayoría |
| **Max 20x ($200/mo)** | $200/mo | Múltiples proyectos + Agent Teams |

**Ahorro:** Subscription 5x vs API directo.

## Cuándo Empezar con Pro

Pro ($20/mo) es suficiente para:
- Uso individual
- 1-2 proyectos activos
- Learning y experimentación

## Cuándo Upgradear a 5x ($100/mo)

Upgrade cuando:
- Hitting límites frecuentemente en Pro
- Múltiples proyectos simultáneos
- Subagents en uso regular
- Context siempre sobre 50%

**Nota:** El upgrade se prorratea — solo pagás la diferencia.

## Cuándo 20x ($200/mo)

- Agent Teams activos
- Múltiples desarrolladores usando la misma cuenta
- Proyectos paralelos de alta intensidad

## Justificación para Teams/Stakeholders

```
API directo:     ~$500/mo
Subscription:   ~$100/mo (5x)
Ahorro anual:   ~$4,800
```

Si alguien pregunta por qué no usar API directo:
- La misma capacidad
- 5x más barato
- Menos overhead de configuración
- Claude Code incluye UI y gestión de contexto

## Context y Costo

El costo no solo depende del modelo sino del context:

- **Sin CLAUDE.md:** ~33K tokens solo para entender proyecto por sesión
- **Con CLAUDE.md:** ~1.2K tokens por sesión
- **27x más barato** con CLAUDE.md optimizado

Esto significa que la misma subscription dura más o equivale a más trabajo efectivo.
