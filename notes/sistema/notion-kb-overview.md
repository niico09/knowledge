# 📚 NotebookLM Knowledge Base

Base de conocimiento construida a partir del workflow documentado por facultad de NYU, Stanford, Case Western, ASU y Northeastern.

## Estructura

| Sección | Descripción |
|---------|-------------|
| 🗺️ Workflow Overview | Los 7 pasos del workflow completo con prompts |
| 📋 Prompts Library | Base de datos de prompts filtrable |
| 🗂️ Notebooks Activos | Registro de todos tus notebooks |
| 🔬 Experimentos & Resultados | Log de lo que probaste y aprendiste |
| 🔗 Integraciones | MCP, Claude Code, Google Classroom |
| 📖 Referencias & Fuentes | Links y fuentes originales |

## Quick Start

1. Empezá por **Workflow Overview** para entender el sistema completo
2. Usá **Prompts Library** para copiar prompts listos para usar
3. Registrá tus notebooks en **Notebooks Activos**
4. Documentá lo que aprendés en **Experimentos & Resultados**
5. Usá **Asset Generator v3** para generar assets
