# Prompts NotebookLM — English B2 Tech Path

Para usar en NotebookLM Studio junto con el Briefing Document. Configuración: **Más preguntas / Difícil** para cuestionarios, **Más tarjetas / Difícil** para flashcards.

---

## The Elements of Style — Strunk & White

### Briefing Document

```
Genera un Briefing Document para Chapters 1-3 de "The Elements of Style" de Strunk & White.
Focus en:
- Las 8 reglas de uso más importantes
- Diferencia entre "which" y "that" (restrictive vs non-restrictive clauses)
- Cómo identificar oraciones wordy
- Casos donde el estilo informal es apropiado vs formal

Estructura:
1. Resumen ejecutivo (5-7 oraciones)
2. Conceptos clave (8-10 puntos)
3. Términos técnicos importantes (glosario)
4. Preguntas guía para profundización
5. Ejemplos de aplicación técnica para cada concepto
```

### Cuestionario

**Configuración:** Más preguntas / Difícil

```
Scope: Analyze ONLY Chapters 1-3 of "The Elements of Style" by Strunk & White. Ignore all other sources.

Genera un quiz para validar dominio profundo de las reglas de escritura técnica en inglés.

Question distribution:
- 40% on clarity and concision: identifying wordiness and rewriting succinctly
- 40% on grammar rules: which/that, active/passive voice, subject-verb agreement
- 20% on style decisions: when formal vs informal is appropriate in technical writing

For each question:
- 4 options (A–D), one correct, three plausible distractors
- After each answer, one-sentence explanation grounded in the chapter source
- Questions must require reasoning about trade-offs or application, not just recall

If a question cannot be grounded in the chapter source, skip it. Do not invent content.
```

### Tarjetas Didácticas

**Configuración:** Más tarjetas / Difícil

```
Scope: Analyze ONLY Chapters 1-3 of "The Elements of Style" by Strunk & White. Ignore all other sources.

Genera flashcards para práctica espaciada. Cada tarjeta debe testear comprensión de un concepto, trade-off, o mecanismo.

Card distribution:
- 40% corrections: front = wordy sentence, back = concise rewrite + rule applied
- 40% rules: front = rule name (1-5 words), back = explanation + technical example
- 20% distinctions: front = "What is the difference between X and Y?", back = precise distinction

Card format:
- Front: 1–5 words maximum
- Back: 2–4 sentences, grounded in chapter source
- No cards for concepts mentioned only once in passing
```

## English Collocations in Use — McCarthy & O'Dell

### Briefing Document

```
Genera un Briefing Document para Units 1-10 de "English Collocations in Use".
Para cada unit incluir:
- 5 colocaciones más útiles del ámbito tecnológico/profesional
- Patrones gramaticales (verb + noun, adj + noun, etc.)
- Errores comunes que hacen estudiantes de nivel B2
- 2 ejemplos de uso en contexto técnico

Lenguaje: Examples en inglés, explicaciones en español para claridad.
```

### Cuestionario

**Configuración:** Más preguntas / Difícil

```
Scope: Analyze ONLY Units 1-10 of "English Collocations in Use" by McCarthy & O'Dell. Ignore all other sources.

Genera un quiz para validar colocaciones en contexto tecnológico y profesional.

Question distribution:
- 40% on verb + noun collocations: choose correct collocate in technical context
- 40% on phrase combinations: identify natural vs unnatural word pairings
- 20% on error detection: spot wrong collocations in technical sentences

For each question:
- 4 options (A–D), one correct, three plausible distractors
- After each answer, one-sentence explanation grounded in the unit source
- Context must be technical/professional scenarios

If a question cannot be grounded in the unit source, skip it. Do not invent content.
```

### Tarjetas Didácticas

**Configuración:** Más tarjetas / Difícil

```
Scope: Analyze ONLY Units 1-10 of "English Collocations in Use" by McCarthy & O'Dell. Ignore all other sources.

Genera flashcards para práctica espaciada de colocaciones.

Card distribution:
- 40% verb + noun: front = verb, back = correct noun collocate + technical example
- 40% phrase building: front = context sentence, back = natural collocation to complete it
- 20% error spotting: front = incorrect collocation, back = correct form + why

Card format:
- Front: 1–5 words maximum
- Back: 2–4 sentences, grounded in unit source
```
