# Learning Path — Inglés B2 Tech (12 Semanas)

## Meta
Reforzar B2 → C1. Enfoque: escritura técnica + conversación tecnológica.

## Sistema de Práctica Espaciada

| Revisión | Cuándo |
|----------|--------|
| 1ra | Día siguiente |
| 2da | 3 días después |
| 3ra | 1 semana después |
| 4ta | 2 semanas después |
| 5ta | 1 mes después |

---

## Semana 1: Fundamentos de Escritura Clara

**Libros:** The Elements of Style (Cap 1-3)

| Día | Tarea |
|-----|-------|
| 1 | Briefing Document: Cap 1-2 (Reglas de uso) |
| 2 | Guía de Reconstrucción → papel: 8 reglas de oro |
| 3 | Flashcards: 15 reglas + ejemplos |
| 4 | Cuestionario: identificá errores en textos técnicos |
| 5 | Práctica: reescribir párrafo técnico (150 palabras) |
| 6 | Audio Overview del capítulo para pre-lectura |
| 7 | Repaso con tarjetas (revisión 1ra) |

**Tarjetas a generar:**
- Regla 1: "Use concrete subject and verb"
- Regla 2: "Put statement in positive form"
- Regla 3: "Use definite, specific, concrete language"
- etc.

---

## Semana 2: Estilo y Voz

**Libros:** The Elements of Style (Cap 4-6) + Microsoft Writing Style Guide

| Día | Tarea |
|-----|-------|
| 1 | Briefing Document: Estilo activo vs pasivo |
| 2 | Guía de Reconstrucción: Cuándo usar voz activa |
| 3 | Flashcards: 20 phrases para voz activa técnica |
| 4 | Cuestionario: rewording pasivo → activo |
| 5 | Práctica: Documentar una función de código en inglés claro |
| 6 | Microsoft Style Guide: Secciones 1-3 (fundamentos) |
| 7 | Repaso (revisión 2da) |

**Tarjetas a generar:**
- "is recommended to" → "we recommend"
- "was implemented by" → "the team implemented"
- "can be divided into" → "divides into"

---

## Semana 3: Concisión y Estilo

**Libros:** The Elements of Style (Cap 7-8)

| Día | Tarea |
|-----|-------|
| 1 | Briefing Document: Wordiness y redundancia |
| 2 | Guía de Reconstrucción: Lista de palabras a eliminar |
| 3 | Flashcards: 25 wordy phrases → concise |
| 4 | Cuestionario: identificar wordiness en docs reales |
| 5 | Práctica: editing challenge (recortar 30% un párrafo) |
| 6 | Lectura: ejemplos de documentación técnica real |
| 7 | Repaso (revisión 3ra) |

**Tarjetas a generar:**
- "in order to" → "to"
- "due to the fact that" → "because"
- "at this point in time" → "now"

---

## Semana 4: Colocaciones I

**Libros:** English Collocations in Use (Unit 1-10)

| Día | Tarea |
|-----|-------|
| 1 | Briefing Document: Qué son colocaciones y por qué importan |
| 2 | Guía de Reconstrucción: 30 colocaciones de technology/work |
| 3 | Flashcards: make/do/build + sustantivos tech |
| 4 | Cuestionario: elegir collocation correcta |
| 5 | Práctica: escribir 5 oraciones técnicas usando nuevas collocations |
| 6 | Audio Overview para reforzar pronunciación |
| 7 | Repaso (revisión 4ta) |

**Tarjetas a generar:**
- make a decision / do research / build a system
- write code / develop software / deploy an application
- solve a problem / fix a bug / debug an issue

## Semana 7: Estructura de Documentos Técnicos

**Libros:** Technical Communication (Markel) Cap 5-8

| Día | Tarea |
|-----|-------|
| 1 | Briefing Document: Cómo estructurar docs técnicos |
| 2 | Guía de Reconstrucción: templates de intro/body/conclusion |
| 3 | Flashcards: 15 connectores para argumentación técnica |
| 4 | Cuestionario: evaluar estructura de un documento real |

## Semana 8: Conversation Técnica I

**Recursos:** Frases de Idioms for Technical Professionals

| Día | Tarea |
|-----|-------|
| 1 | Briefing Document: Idioms para explicar problemas técnicos |
| 2 | Guía de Reconstrucción: 20 idioms con ejemplos tech |
| 3 | Flashcards: 20 idioms para troubleshooting conversations |
| 4 | Cuestionario: usar idiom correcto en contexto |
| 5 | Práctica oral: explicar un bug como si fuera a un compañero |
| 6 | Práctica: role-play de code review en inglés |
| 7 | Repaso (revisión 3ra) |

**Tarjetas a generar:**
- "hit a snag" (encontrar un obstáculo)
- "drill down into" (investigar a fondo)
- "get something off the ground" (lanzar proyecto)
- "the bottom line" (lo esencial)
- "wrap my head around" (entender algo difícil)

## Semana 9: Conversation Técnica II

**Recursos:** Talk Like a Pro + Idioms

| Día | Tarea |
|-----|-------|

| 2 | Guía de Reconstrucción: 25 phrases para presentar/demostrar |

**Libros:** Microsoft Writing Style Guide + ejemplos reales

|-----|-------|

- Microsoft Writing Style Guide (web)
- Technical Communication - capítulos seleccionados

### Notebook 2: "English Collocations & Idioms"
- English Collocations in Use
- Idioms for Technical Professionals

### Notebook 3: "Conversation & Speaking"
- Audios propios grabados
- Transcripciones de podcasts técnicos (Lex Fridman, etc.)
- Notas de phrases usadas

## Daily Routine Sugerida (30 min)

```
Mañana (15 min):
  - 5 flashcards nuevas (revisión espaciada)
  - 10 min lectura de 1 capítulo o 3 páginas

Tarde/Noche (15 min):
  - Práctica: 1 exercise del libro activo
  - ó 5 min speaking practice (grabar)
  - ó 10 min writing exercise

Semanal:
  - 1 Audio Overview como pre-lectura
  - 1 Cuestionario de diagnóstico
```
